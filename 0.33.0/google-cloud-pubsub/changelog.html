pubsub: respect total expiration while modacking (#2715)

Previously we use total expiration only to decide whether to send
new modacks. If we send modacks close to the expiration, we might
extend the lease beyond the total expiration time.

This commit shortens the lease when we approach the expiration time
so we never go over.
